# AlAndaLang

A very serious programming language, made on top of Andalucia's funny spanish pronunciaton.

### 🤔 Examples
Simple calculation:
```
(* 3 3 (+ 2 2))
(js 3+3)
```

### 🧾  Usage

Running a file:
```
$ npm start file
```

### 🤖 Development

Clonning the repository:
```
$ git clone https://github.com/LeonAlvarez/AlAndaLang.git
```

Installing the dependencies:
```
$ npm install
```

Runing a test:
```
$ npm run test
```
